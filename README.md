## 🌐 Add Proxies

The bot supports per-account proxy setup. This helps each account use its own connection path.

To add a proxy:
1. Open the account profile
2. Find the proxy field
3. Enter the proxy address
4. Add the port
5. Add the username and password if needed
6. Save the profile
7. Test the connection if the app offers a test button

Proxy format examples:
- IP:Port
- Username:Password@IP:Port

Keep one proxy matched to one account. That makes setup easier to manage.
